# BookAnalyzerAgent 
**BookAnalyzerAgent** - это телеграмм-бот, который помогает анализировать книги любого жанра. Просто отправьте ему название, и он напишет краткую, но содержательную рецензию. Подходит как для художественной, так и для нон-фикшн литературы.
Проект использует **Misrral Ai** для генерации структурированного анализа и **DuckDuckGo** Search для поиска актуальной информации в интернете.

# Возможности
- Анализ книг по запросу (название + автор)
- Поддержка литературы любого жанра
- Автоматическое определение жанра и выбор соответствующиего шаблона анализа
- Поиск информации через DuckDuckGo (без Api-ключей)
- Генерацию структурированной рецензии с помощью Mistral Ai

# Технологии
- Язык: Python 3.9
- Основные библиотеки:
    - aiogram — для работы с Telegram API.
    - requests — для HTTP-запросов к Mistral AI.
    - duckduckgo-search — для бесплатного поиска информации.
    - python-dotenv — для загрузки переменных окружения.
    - aiohttp — для веб-сервера (необходимо для хостинга на Render)
- Mistral AI API (модель mistral-large-latest)
- DuckDuckGo Search API (бесплатный поиск без ключей)



# Постоянная работа
Бот настроен для бесплатного хостинга и работает без переюоев благодаря связке двух сервисов:
1. Render (Web Service): Код бота запущен на Render как веб-сервисю Это позволяет ему быть доступным в интернете 24/7 на бесплатном тарифе.
2. UptimeRobot (Мониторинг): Бесплатный тариф Render имеет одну особенность - он переходит в спящий режим после примерно 15-20 минут бездействия, чтобы экономить ресурсы. При первом оращении после сна бот будет отвечать с задержкой 30-60 секунд. Чтобы этого избежать мы используем UptimeRobot - сервис, который пингует адрес вашего бота на Render каждые 10 минут. Благодаря этому Render видит постоянную активность и не выключает сервис.
